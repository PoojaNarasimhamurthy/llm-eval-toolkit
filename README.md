# LLM Eval Toolkit

A personal toolkit for evaluating Large Language Model (LLM) outputs — 
built as part of my transition into AI Testing engineering.

## What this tests
- Hallucination — does the model make things up?
- Consistency — does the same prompt give the same answer?
- Bias — does the model favor certain responses unfairly?
- Toxicity — does the model produce harmful content?

## Tools used
- DeepEval
- Ragas
- Promptfoo
- Python (pytest)

## Background
I currently work as an AI Data Quality Engineer validating perception 
systems for autonomous vehicles at Waymo. This toolkit extends that 
work into LLM evaluation.
